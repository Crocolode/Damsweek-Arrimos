# Damsweek-Arrimos

Projeto de TCC em engenharia civil para dimensionamento de muros de arrimo (flexão e gravidade).

## Arquivos principais

- `Dimensionamentos.py`: script principal para cálculo e dimensionamento.

## Dependências

- `matplotlib`
- `tkinter` (biblioteca padrão do Python para interfaces gráficas)

## Como usar

1. Crie e ative um ambiente virtual (opcional, mas recomendado):
   ```powershell
   python -m venv .venv
   & .venv\Scripts\python.exe -m pip install --upgrade pip
   & .venv\Scripts\python.exe -m pip install -r requirements.txt
   ```

2. Execute o script:
   ```powershell
   & .venv\Scripts\python.exe Dimensionamentos.py
   ```

Se preferir, você também pode executar diretamente com o Python do sistema:
```powershell
python Dimensionamentos.py
```
